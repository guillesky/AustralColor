AustralColor

Underwater Image and Video Color Correction

Download

Features

  • OpenCV based underwater color correction
  • Image and video processing
  • FFmpeg integration
  • Concurrent processing support
  • Multi-language interface
  • Portable Windows executable

About

AustralColor is a desktop application designed to improve underwater photos and videos by restoring natural colors lost underwater.

Developed in Java using OpenCV and FFmpeg.
